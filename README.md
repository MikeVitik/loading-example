## Storybook паттерны загрузки данных

Этот мини-проект демонстрирует несколько подходов к загрузке данных в React 18 + TypeScript
для сценария «список + панель деталей» с защитой от гонок запросов:

- наивный подход (проблема гонок);
- guard через счётчик `requestId`;
- использование `AbortController` для отмены запросов;
- `useTransition` для мягкого переключения выбора;
- `useDeferredValue` для разгрузки интерфейса;
- `debounce` в сочетании с guard для уменьшения числа запросов.

Для симуляции сети проект использует `fetch` (предполагается обёртка через `fetch-network-simulator`
в конфигурации Storybook).
